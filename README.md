- Atualmente possui:

- controller de tarefas .js
- route de tarefas . js
- banco de dados para tarefas . sql
- acesso ao banco por db . js


Você consegue criar tarefas novas inserindo o nome e a descrição delas. Estas tarefas novas automaticamente recebem o status de "Pendente" e vão para uma coluna na interface que lista todas as tarefas pendentes. Depois de criar, você pode tanto excluir quanto editar a tarefa. Ao clicar em remover, ela é simplesmente removida do banco de dados e da interface. Ao clicar em editar, você tem as opções de editar tanto o nome e descrição quanto o status dela. Caso você mude o status para "Em progresso" ou "Concluída", a tarefa em questão é movida para a coluna selecionada.
